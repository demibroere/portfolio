KawaiiBox Overview

Voor het vak frontend development moesten wij een website kiezen om na te bouwen, van de gekozen website moest je dan 2 pagina's nabouwen. Ik heb gekozen voor de website van Kawaiibox, omdat dit een kleurrijke website is en ook erg vrolijk. Er kwamen dingen in voor waar ik nog nooit mee te maken had gehad en ik vond het dan ook erg leuk om er mee aan de slag te gaan. Bij de opdracht kon je kiezen of je voor surface plane ging of voor het responsive maken van de website. In eerste instantie had ik ervoor gekozen om mij te focussen op de surface plane, maar omdat ik ook graag wilde weten hoe je een website volledig responsive kon maken heb ik dat ook

Bij de opleiding MBO 4 ICT heb ik al te maken gehad met HTML en CSS en dat vond ik toen al super leuk om te doen. Hier heb ik de basis al geleerd van coderen.

Ik heb geleerd hoe je een website responsive kan maken, ook heb ik geleerd hoe ik met grid en flexbox kan werken en dat dit dingen een stuk makkelijker kan maken. Tijdens dit project heb ik ook meer geleerd over het werken met keyframes, de hover en het belang van het accessible maken van de website. In de toekomst hoop ik nog veel meer te kunnen leren over HTML en CSS, zodat ik nog betere websites kan neerzetten. Voor dit project heb ik ook het minimale aan JavaScript kunnen gebruiken.
